# bandtango-mobile

Mobile apps for BandTango.

## React Native App

A new Expo + React Native project is available in [bandtango-app](bandtango-app) with a playlist management flow inspired by the wireframe prototype:

- Prototype: https://www.figma.com/proto/Qaj4MSzIvanudGOYuCag0Y/BandTango-Wireframe?node-id=1-40
- Home screen with playlist list
- Create playlist screen
- Playlist detail screen with add/remove songs

## Run Locally

1. Install Node.js 20+ (Expo SDK 54 requirement).
2. From the app folder:

```bash
cd bandtango-app
npm install
npm run ios
```

You can also use `npm run android` or `npm run web`.
